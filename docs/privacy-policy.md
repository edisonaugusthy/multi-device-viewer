# Multi Device Viewer Privacy Policy

Last updated: May 9, 2026

Multi Device Viewer is a local-first Chrome extension for previewing websites across device viewport profiles and capturing responsive testing screenshots.

## Data Collection

Multi Device Viewer does not collect, sell, transfer, or share user data with third parties.

The extension does not use analytics, telemetry, advertising trackers, remote logging, or a backend service.

## Local Data

The extension stores preferences locally in Chrome storage on the user's device. This may include favorite devices, recent device choices, custom viewport settings, annotations, and extension state.

Screenshots, annotations, URLs, and settings are processed locally in the browser. Exported screenshots are saved only when the user chooses to download them.

## Website Access

The extension can load user-selected website URLs inside local preview frames so users can test responsive layouts. This access is used only for the extension's single purpose and is not sent to a remote service.

## Remote Code

Multi Device Viewer does not execute remotely hosted JavaScript or WebAssembly. All extension code is packaged with the submitted extension.
