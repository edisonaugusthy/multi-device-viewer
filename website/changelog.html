Changelog

A record of the improvements that make responsive review faster, clearer, and more reliable.

0.2.0 — Streamlined workspace

  • Clean-canvas mode hides the sidebar, main toolbar, and device toolbars while preserving resize and restore controls.
  • Correct emulator night mode without filtering or recoloring previewed pages.
  • Additional 2025–2026 Android device frames with corrected viewport and camera geometry.
  • Explicit navigation sync and a live source-tab recording indicator.
  • Responsive AI fix-prompt preview with fully optional issue context and stronger verification guidance.
  • Physical-device handoff for LAN-ready development pages.

0.1.5 — Everyday workflow

  • Faster device switching, saved sets, a resizable workspace, reliable linked scrolling, and design comparisons.
