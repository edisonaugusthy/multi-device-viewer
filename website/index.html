COMMON QUESTIONS

Good to know.

Is this a real mobile browser emulator?+

It is a fast responsive development aid inside Chrome or Firefox. It reproduces viewport dimensions and useful device presentation, while final release testing should still cover physical devices and target browsers.

Can it test localhost?+

Yes. Open your local development page in Chrome or Firefox, then launch the extension. For physical-device handoff, expose the dev server on your LAN and use your computer's network address instead of localhost.

Why can’t some websites load?+

Some applications prevent embedding through authentication rules or frame protection. The viewer reports the limitation and provides an open-in-tab fallback.

Does the AI prompt feature upload my code?+

No. It formats a prompt locally for you to copy into the coding assistant of your choice. The extension does not send the prompt or your code to an AI service.
